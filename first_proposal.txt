Project Proposal: Visualizing Fertility Rate and Life Expectancy Trends

1. Research Question
How have global fertility rates and life expectancy evolved from 1964 to 2013, and how can an interactive visualization reveal patterns, correlations, and regional disparities in this relationship?

2. Project Description
This project aims to create an interactive dashboard to explore the dynamic between national fertility rates and life expectancy over five decades. Using the structured, publicly available dataset from Gapminder.org, I will develop a clear narrative about global health and demographic changes.

3. Learning Goal and New Element
The primary technical goal is to learn and implement the Bokeh library in Python to build rich, web-based interactive visualizations. Bokeh has not been covered in the course and is perfectly suited for creating linked charts (e.g., animated scatter plots, time series) with hover tooltips and selection filters, allowing users to explore data by country, region, and year.

4. Execution
After data cleaning, I will use Bokeh to design a multi-plot dashboard. The main view will be an animated scatter plot where each point represents a country, with axes for fertility and life expectancy, sized by population. A timeline slider will animate changes. Linked charts will show temporal trends for selected countries. The code will be managed and shared via GitHub.
